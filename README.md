<h1 align="center">📱 Communication Platform (PWA)</h1>
<p align="center">A Progressive Web App enabling real-time communication and seamless Android ↔ Desktop device linking.</p>

---

## 🚀 Features

- 📞 **Video Call** functionality using WebRTC
- 💬 **Messaging** system for real-time text communication
- 🔗 **Device Linking** between Android and Desktop
- 📁 **File Sharing** (Images, GIFs, Word Docs, PDFs)
- 📷 **Image Sharing**
- 🧩 Built as a **Progressive Web App (PWA)**

## 🛠️ Tech Stack

- **Frontend**: React, Redux, TailwindCSS
- **Backend**: Node.js, Express
- **Realtime**: WebRTC, Socket.io
- **File Sharing**: Base64 / Blob Conversion
- **Build Tools**: Vite / Webpack
- **Hosting**: Firebase / Vercel (Optional)
